Hvilke programmer og nettsider trenger du til IT på Sandvika?

Forsiden

Atom

Atom er et gratis open-source program som man kan bruke til å kode for macOS, Linux og Windows. Programmet er utviklet av GitHub, noe som gjør at man enkelt kan legge ut arbeidet sitt.

Atoms nettsider

Adobe-pakken

I IT bruker vi Adobe Creative Cloud en god del. Der har de flere nyttige programmer, slik som Photoshop og Illustrator, som kan brukes til bilderedigering og for å lage illutrasjoner, logoer og så videre.

Adobe Creative Clouds nettsider

W3Schools

Hos W3Schools finner du alt mulig av eksempler og veiledning til forskjellige programmeringsspråk. Det kan brukes under eksamen. Veldig nyttig!

W3Schools nettsider

GitHub

GitHub er en web-basert vertstjeneste, som vi bruker i IT-faget til å legge ut arbeidet vårt på nett.

GitHubs nettsider
